# 🔗 راهنمای اتصال API باشگاه مشتریان اسمارت به وبسایت Cake Shop

## نحوه کار

### فلوچارت کلی:
```
مشتری موبایل/کد باشگاه وارد میکنه
            ↓
   ① احراز هویت (Login) → دریافت Token
            ↓
   ② استعلام کد باشگاه (GetCustomerInquiry)
            ↓
      نمایش اعتبار/تخفیف به مشتری
            ↓
   ③ استفاده از اعتبار (UsedCredit)
            ↓
      اعمال تخفیف روی فاکتور
            ↓
   ④ ارسال فاکتور به سرور (Factor)
```

### فایل‌ها:
- `smart-api.js` — ماژول JavaScript برای ارتباط با API اسمارت
- `club-widget.html` — ویجت باشگاه مشتریان (قابل اضافه شدن به index.html)
- `club-widget.css` — استایل ویجت
- `server-proxy.js` — پراکسی سرور Node.js (برای امنیت توکن)
- `integration-guide.md` — راهنمای قدم به قدم

## ⚠️ نکته مهم
چون API اسمارت از سمت سرور کار میکنه (Bearer Token)، نباید Token رو توی فرانت‌اند ذخیره کنی.
باید یه پراکسی سرور Node.js بسازی که درخواست‌ها رو از فرانت به API اسمارت ارسال کنه.
